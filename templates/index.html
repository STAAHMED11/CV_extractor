About This Tool

This application extracts structured information from CVs/resumes using open-source Large Language Models (LLMs) running on Ollama.

Features:

  • Extract name, contact details, education, skills, and experience from CVs
  • Support for both text-based and scanned PDF documents
  • Compare extraction results across different LLMs
  • Evaluate extraction accuracy with precision and recall metrics

Available models: LLaMA 3, Mistral, and Phi.
